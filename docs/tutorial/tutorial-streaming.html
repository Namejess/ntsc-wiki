Guide de la Régie de Streaming

Ce guide vous accompagne dans la mise en place et l'utilisation de votre régie de streaming.

Configuration Matérielle​

Branchements​

  1. Caméra

    • Connectez la caméra à l'entrée HDMI de la carte de capture
    • Assurez-vous que la caméra est en mode sortie HDMI

  2. Microphone

    • Branchez le microphone XLR dans la console de mixage
    • Connectez la sortie de la console au PC via USB ou interface audio

  3. Ordinateur

    • Connectez la carte de capture via USB
    • Vérifiez que tous les périphériques sont reconnus

Configuration Logicielle​

OBS Studio​

  1. Installation

    • Téléchargez OBS Studio depuis le site officiel
    • Installez le logiciel en suivant l'assistant

  2. Configuration des Sources

    • Ajoutez une source "Capture Vidéo" pour la caméra
    • Ajoutez une source "Capture Audio" pour le microphone
    • Configurez les scènes selon vos besoins

  3. Paramètres de Streaming

    • Configurez la qualité de streaming (bitrate recommandé : 6000 kbps)
    • Sélectionnez votre plateforme de streaming (Twitch, YouTube, etc.)
    • Testez la connexion avant de lancer le stream

Gestion du Son​

Mixage Audio​

  1. Niveaux de Son

    • Ajustez le gain du microphone pour éviter la saturation
    • Vérifiez les niveaux dans OBS (idéalement entre -12 et -6 dB)
    • Activez le filtre de bruit si nécessaire

  2. Égalisation

    • Utilisez un égaliseur pour améliorer la qualité du son
    • Réduisez les fréquences basses pour éviter les résonances
    • Augmentez légèrement les aigus pour plus de clarté

Bonnes Pratiques​

  1. Avant le Stream

    • Vérifiez tous les branchements
    • Testez le son et l'image
    • Préparez vos scènes et transitions

  2. Pendant le Stream

    • Surveillez les niveaux audio
    • Vérifiez la qualité de la connexion
    • Gardez un œil sur le chat

  3. Après le Stream

    • Sauvegardez vos paramètres
    • Débranchez proprement le matériel
    • Faites un compte-rendu des problèmes rencontrés

Dépannage​

Problèmes Courants​

  1. Pas de Son

    • Vérifiez les branchements
    • Contrôlez les niveaux dans OBS
    • Vérifiez les paramètres Windows

  2. Image Noire

    • Vérifiez la source vidéo
    • Redémarrez la carte de capture
    • Mettez à jour les pilotes

  3. Latence

    • Réduisez la qualité du stream
    • Vérifiez votre connexion internet
    • Fermez les applications inutiles
